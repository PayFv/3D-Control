# 3D Hand Control Particles

A real-time interactive 3D particle system controlled by hand gestures, built with Three.js and MediaPipe.

## Features

-   **Hand Tracking**: Control particles with hand gestures (Open to expand, Close to shrink).
-   **Templates**: Switch between Heart, Flower, Saturn, Buddha, Fireworks.
-   **Text Mode**: Type any text to generate particle shapes.
-   **Customization**: Adjust particle colors in real-time.

## Installation

1.  Install dependencies:
    ```bash
    npm install
    ```

## Running

1.  Start the development server:
    ```bash
    npm run dev
    ```
2.  Open your browser at `http://localhost:5173`.
3.  Allow camera access when prompted.

## Usage

-   **Expansion**: Open your hand and stretch your fingers to make particles explode outwards.
-   **Shrink**: Close your hand into a fist to shrink the shape.
-   **Shape Selection**: Use the UI panel to pick shapes or enter text.
